Neobazaar
========================

Free classifieds web application

-- work in progress --


### Install with Composer
 ```
{
  "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/kaiohken1982/Neobazaar.git"
        }
    ],
    "require": {
        ......,
        "razor/watermarker" : "dev-master"
        "razor/cropper" : "dev-master"
        "razor/thumbnailer" : "dev-master"
    }
}
 ```
